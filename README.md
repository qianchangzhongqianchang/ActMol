# ActMol
Active Perception-Guided 3D Molecular Representation Learning for Drug Discovery
## Project Structure

| File/Folder | Description |
|---|---|
| `data/` | Dataset files |
| `DDI/` | Drug-drug interaction related files |
| `DTI/` | Drug-target interaction related files |
| `models/` | Model definitions |
| `main.py` | Main running script |
| `feature.py` | Feature extraction script |
| `TAID_sdf.py` | SDF file processing script |
| `ply.py` | PLY file processing script |

## Requirements

The code is implemented in Python. The main dependencies include:

```text
python >= 3.8
```
